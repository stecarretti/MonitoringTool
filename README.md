# MonitoringTool
## This tool exploits the functionalities of Cloud Debugger API in order to create metrics about how, how much, when and where any web application deployed with GCP is used
## GCP's Services List
- Compute Engine
- App Engine
- Cloud Functions
- Cloud Firestore
- Cloud Endpoints
